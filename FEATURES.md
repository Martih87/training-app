# Lightweights — Planned Features

## 1. User-Linked Exercises
Save registered exercises per Google account so each user has their own persistent exercise library.

## 2. Exercise Suggestions
Suggest exercises to users based on what others have logged — a community-driven recommendation feature.

## 3. Add "Landmine Barbell Rows"
Add **Landmine Barbell Rows** as a built-in exercise with a default weight of **30 kg**.

## 4. Edit Sessions
Allow users to edit previously logged training sessions (update exercises, sets, weights, reps, or delete entries).

## 5. Data Export
Export exercise and session data for use in external apps or services (e.g. CSV, JSON, or API-based data pull).
